What is HTML ?!!!

  • HISTORY OF HTML

    • HTML is the language for describing Web documents(web pages). HTML stands for HYper Text Markup Language. A markup language is a set of markup tags.HTML documents are described by HTML tags. Each HTML tag describes different document content.
  • Background History
